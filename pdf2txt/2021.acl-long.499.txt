we set n = 2 for roc and n = 5 forwp.
(4) semantic repetition (sr-n): the metricÔ¨Årst computes the average top-n sentencebertsimilarity between any two sentences in each gen-erated text, and then averages the results as the Ô¨Ånalscore.
we set n = 1 for roc and n = 10 for wp.
